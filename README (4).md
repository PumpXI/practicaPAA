# Simulacro PAA UCR — Adecuación Tiempo Extendido

Plataforma de simulacro para la Prueba de Aptitud Académica (PAA) de la Universidad de Costa Rica, modalidad con adecuación de tiempo extendido (2 horas 20 minutos).

## Características

- **45 ítems oficiales** del folleto *60-10 preguntas y respuestas de práctica* (IIP-UCR)
- **Cronómetro de 2h 20min** que corre de manera continua y no se puede pausar
- **Estado persistente:** si se cierra o recarga accidentalmente la página, el examen se reanuda desde donde se dejó (con el tiempo correcto)
- Mapa visual de ítems con navegación libre
- Cálculo automático del promedio de admisión UCR (escala 200–800) usando la fórmula oficial del proceso 2025–2026 con nota de presentación de 78
- Revisión detallada de respuestas al finalizar con filtros por correcto / incorrecto / omitido

## Uso

Abrir `index.html` directamente en el navegador. No requiere servidor ni dependencias externas.

## Publicar en GitHub Pages

1. Subir este repositorio a GitHub
2. Ir a **Settings → Pages → Branch: main → / (root)**
3. Compartir el enlace generado (`https://usuario.github.io/nombre-repo/`)

## Fuente

Ítems extraídos del folleto oficial *"60-10 preguntas y respuestas de práctica para la Prueba de Aptitud Académica"*, Instituto de Investigaciones Psicológicas, Universidad de Costa Rica (paa.iip.ucr.ac.cr).
